NuMuseum API

About

The NuMuseum API returns information about the items in the NuMuseum virtual collection. This information includes descriptions, types (eg sculpture, architecture, painting etc), mediums, dimensions, artists and more. It also returns information about the artists and departments. The API also enables user registration, user-generated lists of favourite works and deleting accounts.
